Installing 7HUD and Information

7HUD is a Team Fortress 2 HUD created by Sevin. 7HUD was originally based of ya_HUD, but over time it became it's own style and direction and became a completely independent HUD.

7HUD is incredibly easy to install and is as simple as placing the folder in "tf/custom/", if you still have troubles with installing the HUD or any of its features, you can view the documentation on the HUD by clicking the button below.

Submitting Bugs and Issues

The submission and tracking of bugs for 7HUD is easily done. They can be submitted to the Github section or posted on the TeamFortress.TV thread. Submitting these errors allows Sevin to continually improve the HUD and quickly find fixes for any issues that may arise with the HUD.
